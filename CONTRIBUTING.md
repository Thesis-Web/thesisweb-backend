
# Contributing

## Quick start
```bash
npm ci
npm run format:check
npm run lint --if-present
npm run build --if-present
```

## Formatting
- Run: `npm run format`
- CI enforces: `npm run format:check`

## Commits
- Keep commits small and descriptive.
- Prefer: `type: short description` (e.g., `docs: clarify nginx rewrite`)
